Recovery (Bricked Flipper)

  1. Unplug USB completely
  2. Hold BACK for 30 s — hardware-level reset
  3. Hold BACK + LEFT for 3 s — soft reboot
  4. Hold BACK + OK for 35 s
  5. At 35 s: release BACK, keep holding OK
  6. While holding OK: plug in USB
  7. Open qFlipper → Repair Firmware

Forces STM32 into hardware boot ROM recovery mode — bypasses any broken firmware entirely.
